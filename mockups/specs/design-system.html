Neotilde — Design system

Single source of truth for color, type, and the small component vocabulary used across every feature mockup. If you're starting a new mockup, copy the :root palette block and reuse the components below.

Palette — "Neon Midnight" (default)

The live default theme. Coral neon on deep blue-black — the orange-red glow of neon tubing against a near-black night ground — with verdigris as the success/tension color. Accent is solid at rest; only the bell glows. Sidesteps salmon-clay (Anthropic), mono-green (ChatGPT), gradient-purple ("AI"), matrix-green and phosphor-amber (terminal stereotypes), ice-blue, and the cyan-magenta excess of CP2077. Bell bronze is retained as a switchable alternate (Pro cosmetic).

Component vocabulary

Small set of reusable pieces every feature mockup pulls from. If you find yourself inventing a new pill or sheet shape, ask first whether one of these does the job — the goal is visual consistency, not novelty.

Keybar slot

Rectangular tap target in the keybar. Bronze hint glyph below the label indicates a long-press action; absence of hint = tap only.

Pill (window / pane)

Rounded slot in the keybar's locked-left section. Shows a current-state label; gestures (tap/swipe/long-press) drive nav.

Banner (top of screen)

Transient by default (auto-dismiss); persistent only in documented exceptions (tmux-crash). Amber = degraded, red = broken.
