semicolyn — Blue-oriented theme candidates

Both concepts keep Neon Midnight's blue-near-black night (#06080F ground, #04060D terminal) and push saturation. Rest is solid — glow is bell-only.

Same shell session in every card so each palette shows in context: prompt, path, git branch, an ok, an error, a warning, a link, and the cursor block.

B · Even-spaced, blue-anchored — a 72° hue ring; meaning wins on the snap

Anchor the ring on blue at 216°, step 72° five times → violet, red, (yellow), green. The pure-math yellow at 72° is nudged to amber (43°) so warning still reads warm — that's the "meaning wins" snap. Every hue is hsl(h 88% 64%)-class: identical saturation/lightness, only hue moves.
